# Source register

The repository uses authoritative standards and project sources where possible: HL7 FHIR, IHE BALP/ATNA, RFC 9162, Trillian, Rekor, Hyperledger Fabric, HAPI FHIR, EUR-Lex, GDPR, ISO public catalogue material and peer-reviewed literature cited in the accompanying manuscript and documentation.

Reference metadata should be checked against the target manuscript before submission, especially DOI formatting and access dates for grey literature.
