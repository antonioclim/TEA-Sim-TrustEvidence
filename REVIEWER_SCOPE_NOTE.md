# Reviewer scope note

This package supports a cautious design-science and reproducibility claim: the TrustEvidence boundary is specified, local reference artefacts execute and selected append-only/Merkle properties are tested under declared assumptions.

It does not support claims of FHIR/BALP conformance, IHE certification, production deployment, clinical validation, legal compliance, a blockchain benchmark, a PostgreSQL benchmark or a full cryptographic proof.
