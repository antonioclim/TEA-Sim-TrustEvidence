# TrustEvidence formal/property validation workstream formal/property kit

This kit contains formal/property validation workstream property-based and bounded executable checks for the A2 Merkle local reference backend. It is an evidence workstream artefact, not a production security certificate and not a public release archive.

## Main command

```bash
python scripts/run_property_checks.py
```

TLA+ and Alloy specifications are included as scaffolding. They were not executed because TLC and Alloy were unavailable in the runtime.
