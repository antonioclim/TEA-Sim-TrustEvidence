# Limitations

1. The checks apply only to the A2 Merkle local reference backend.
2. Property-based testing covers generated cases, not all possible states.
3. The bounded executable model is finite-scope.
4. TLA+ and Alloy were not executed.
5. Receipt digest is a local self-consistency commitment, not a signature or external witness.
6. The results do not establish production security, legal compliance, clinical validation or FHIR/BALP conformance.
