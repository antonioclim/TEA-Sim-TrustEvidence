# Output manifest

- `calibrated_scenarios/calibrated_scenarios.csv`: C1-C4 workload descriptors and derivations.
- `calibrated_scenarios/PARAMETER_REGISTER_CALIBRATED.csv`: source metadata, computed parameters and assumptions.
- `calibrated_scenarios/synthea_stats.csv`: Synthea metadata-derived status file.
- `calibrated_scenarios/bigideas_stats.csv`: BIG IDEAs metadata-derived status file.
- `calibrated_scenarios/mimic_fhir_stats.csv`: MIMIC-on-FHIR metadata-derived status file.
- `benchmark_outputs/calibrated_a2_summary.csv`: local capped A2 Merkle passage results.
- `benchmark_outputs/raw_runs/`: raw local A2 run summaries.
- `smoke_outputs/`: parser smoke outputs from bundled toy samples only.
- `metadata/VERIFIED_PUBLIC_SOURCE_PARAMETERS.csv`: source metadata and assumptions used to construct scenarios.
