# Workload descriptors

The executable reference workload uses deterministic synthetic monitoring-accountability events and three bounded tree sizes: 128, 512 and 2,048 leaves. Twelve repetitions are executed for each size, yielding 32,256 events in the retained full run. Thirty-two receipts are sampled per repetition and one re-signed proof-path mutation is applied per repetition.

The retained raw event and timing rows were produced by the integrated workload script under run identifier `cmpb-workload-reference-001`. Timing values are descriptive single-host measurements. They are not a comparative benchmark, a production-capacity estimate or evidence of clinical responsiveness.
