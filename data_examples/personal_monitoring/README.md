# Synthetic personal-monitoring examples

These fixtures contain synthetic monitoring-accountability metadata for object registration, access, consent transition, provenance transformation, disclosure, aggregation and failure events. They contain no patient identifiers, raw physiological values or restricted clinical records.

The Ed25519 values, payload commitments and local receipt material are deterministic test fixtures. Structural schema checks, canonicalisation, signatures, inclusion proofs and retained-checkpoint consistency are evaluated by separate executable experiments. The fixtures are not operational credentials and must not be reused outside testing.
