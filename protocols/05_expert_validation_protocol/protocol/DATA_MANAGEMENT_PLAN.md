# Data management plan

## Status

Protocol only. This file does not document completed data collection.

## Separation

Recruitment contact details must be stored separately from survey responses. Analysis files must use pseudonymous participant identifiers only.

## Storage

Use institutionally approved encrypted storage. Do not put identifiable expert contact data, raw free text or participant-level files into GitHub, Zenodo or manuscript supplements unless the ethics approval, consent language and anonymisation assessment explicitly permit it.

## Publication

Report aggregate statistics and anonymised comment summaries. Screen free text for accidental identifying details before quotation or paraphrase.
