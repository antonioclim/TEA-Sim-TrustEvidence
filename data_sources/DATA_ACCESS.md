# Data-source access boundary

The integrated v2.1.0 execution uses no downloaded external clinical dataset. Every event passed through the implementation is synthetic.

External sources have only two bounded roles:

1. a previously reported aggregate-only single-participant CGM extraction used during scenario design, with no raw values retained here;
2. public documentation used to inform plausible object heterogeneity and bounded scenario parameters.

Live access, licence and version information must be reverified in the reference-verification phase before manuscript submission.
