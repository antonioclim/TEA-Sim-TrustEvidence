# C3 materialisation trigger

This immutable marker triggered the controlled GitHub Actions job that executes the complete Route C C3 FHIR toolchain, retains the compact validation evidence, rebuilds the repository integrity manifests and commits the validated outputs to `revision/jcis-route-c`.

The marker itself is not scientific evidence. The authoritative evidence is the generated case corpus, validation reports, tool-version and checksum records, workflow run, materialisation commit and subsequent full release-check.
